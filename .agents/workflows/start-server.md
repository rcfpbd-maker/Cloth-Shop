// turbo-all

1. Open Git Bash and navigate to the project directory:
```bash
cd /d/project/laravel
```

2. Start the PHP built-in development server with the correct PHP binary and php.ini config:
```bash
/c/php-8.3.29-Win32-vs16-x64/php.exe -c /d/project/php.ini -S 127.0.0.1:8000 -t public
```

3. Open your browser and go to: http://127.0.0.1:8000
